OrbState enum:
  idle, listening, thinking, speaking

ConversationOrb widget (StatefulWidget):
  - size parameter (default 110px)
  - state parameter (OrbState)
  - 4 AnimationControllers: rotation, pulse, wave, burst

Particle System:
  - 20 particles total: 15 on 3 orbital rings (tilted at 0.25, 1.05, 1.85 rad) + 5 scattered
  - Each particle: orbitTilt, orbitRadius, startAngle, speedFactor, size
  - Built once with Random(seed: 73) for deterministic layout
  - Projected from 3D orbital coordinates to 2D using trig
  - Depth-sorted (back-to-front) for correct 3D layering
  - Depth-based opacity (0.25 → 1.0) and size (50% → 100%) scaling

_OrbPainter (CustomPainter):
  1. Wide ambient glow (RadialGradient, 4x core radius)
  2. Tight mid glow (2.2x core radius)
  3. Speaking wave rings (4 concentric rings pulsing outward) — speaking only
  4. Back-layer particles (z < 0)
  5. Core sphere (RadialGradient: highlight → color → shadow)
  6. Specular highlight (small bright circle, upper-left)
  7. Front-layer particles (z >= 0)
  8. Burst ring (state-activation flash)
