── HOW TO TEST ───────────────────────────────────────────────────

1. Build and install:
   cd C:\CommuteCoder\gerald_app
   flutter run

   Or to install the debug APK:
   flutter install

2. App Icon:
   - Check Android launcher — should show gerald_logo.png
   - Check app switcher — should show same icon
   - Look for blue glow in splash screen

3. UI / UX:
   - Verify splash screen: expanding ring, glow, progress indicator
   - Home screen: premium AppBar with two-line Gerald header
   - Status panel: animated dot, correct colors per state
   - Mode selector: tap A ↔ B, verify sliding indicator animates
   - Empty state: centered logo with glow + mode hints

4. Voice Controls:
   - Mode A: press and hold the blue circle mic → turns red, "LISTENING"
   - Mode A: release → sends recognized speech
   - Mode B: tap "CONVERSATION" mode → auto-starts listening
   - Mode B: speaking: expanding ripple ring visible

5. Voice Response (TTS):
   - Go to Settings → enable "Speak Responses"
   - Send a voice command
   - Gerald should speak response aloud
   - Status dot turns purple, "SPEAKING" label appears
   - STOP button appears in utility row and status panel
   - Tap STOP → TTS stops immediately

6. Backend URL:
   - Settings → enter your PC's LAN IP (e.g. http://192.168.1.x:8000)
   - Tap SAVE & RECONNECT
   - Status dot should change from red OFFLINE to blue/green

7. Conversation mode + TTS:
   - Enable both Conversation Mode and Speak Responses in Settings
   - Send a voice command
   - Gerald speaks → when done → auto-resumes listening (Mode B)
