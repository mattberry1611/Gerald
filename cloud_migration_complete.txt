DEPLOYMENT STEPS (quick reference)
------------------------------------
  1. Create DigitalOcean Droplet (Ubuntu 22.04, $6/mo)
  2. scp gerald_bridge.py build_verifier.py multi_ai_router.py root@IP:/app/
  3. scp cloud_config.json root@IP:/app/
  4. Install: apt install python3-pip nginx certbot python3-certbot-nginx
  5. pip3 install fastapi uvicorn
  6. Create /etc/systemd/system/gerald.service → systemctl enable --now gerald
  7. Configure nginx reverse proxy for your domain
  8. certbot --nginx -d gerald.yourdomain.com
  9. Update Flutter Settings → Base URL to https://gerald.yourdomain.com

OR with Docker:
  python cloud_migration/cloud_setup.py --docker
  docker run -d -p 8000:8000 gerald-bridge:latest
