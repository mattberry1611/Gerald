JAVASCRIPT BEHAVIOUR NOTES
---------------------------
- Space bar toggles mic (keydown listener on document)
- /ask response populates #response-body; enables Approve & Discard buttons
- Approve → POST /send-to-claude-code → status bar flashes green
- Discard → clears #response-body; disables buttons; resets to idle
- Activity log entries: prepend <li> with [hh:mm:ss] icon message
- project-override select onchange → POST /set-project or query-param on next /ask
- Status bar colour driven by CSS class on #status-bar (status-idle / listening / processing / approved / error)
