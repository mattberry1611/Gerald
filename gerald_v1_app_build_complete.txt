HOW TO RUN THE APP
─────────────────────────────────────────────────────────────────

STEP 1 — Install Flutter (if not already installed):
  https://docs.flutter.dev/get-started/install/windows
  Confirm: flutter doctor

STEP 2 — Fetch dependencies:
  cd C:\CommuteCoder\gerald_app
  flutter pub get

STEP 3 — Start the backend on your PC:
  cd C:\CommuteCoder
  uvicorn gerald_bridge:app --host 0.0.0.0 --port 8000

STEP 4 — Connect Android phone:
  Enable Developer Options → USB Debugging
  Plug in via USB
  flutter devices  (confirm phone appears)

STEP 5 — Run:
  flutter run

  Or build a debug APK for sideloading:
  flutter build apk --debug
  APK: build\app\outputs\flutter-apk\app-debug.apk

STEP 6 — First time on phone:
  Open Settings → change URL to http://192.168.1.XXX:8000
  (your PC's LAN IP — find with ipconfig on Windows)
  Tap Save & Reconnect
  Status dot should go from red → grey (idle)
